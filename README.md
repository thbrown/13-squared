## js13k 2024

I made this game for js13k 2024.

## Prerequisites

1. install node
2. install npm

## To run

`npm install`
`npm run build-dev`

Then open `./dist/index.html` in your browser

## Other Scripts

```
  "build-prod": "webpack --mode=production --min",
  "build-dev": "webpack --mode=development",
  "build-watch": "webpack --mode=development --watch"
```
